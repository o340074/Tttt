# AdVault — живой прототип

`index.html` — самодостаточный интерактивный прототип и style-guide платформы
**AdVault** в фирменном стиле (premium-dark «Aurora», flash/glow-эффекты,
премиальные анимации).

## Как открыть
- Просто открой `index.html` в браузере — всё работает офлайн (нет внешних
  шрифтов, картинок и скриптов, всё инлайн).
- Либо смотри опубликованную версию как Artifact на claude.ai.

## Что внутри (кликабельные экраны)
| Экран | Что демонстрирует |
|-------|-------------------|
| **Storefront** | Hero с анимированным Aurora-фоном, trust-бар, категории, популярные товары |
| **Catalog** | Панель фильтров, чипы, сетка карточек с hover-glow и sheen-бликом |
| **Product** | Выбор варианта (SKU) с count-up цены, «Buy now» → success-flash + confetti + toast |
| **Wallet** | Баланс, форма пополнения, генерация QR/адреса, «Simulate confirmation» → aurora-flash + count-up |
| **Dashboard** | KPI-карточки со спарклайнами, таблицы заказов, «Vault» с копированием секретов (flash) |
| **Components** | Палитра, кнопки (все варианты + loading), бейджи, инпуты, тогглы, тосты, flash |

## Интерактив
- Переключение экранов через табы (скользящий градиентный индикатор).
- Тёмная/светлая тема — кнопка ◐ в шапке.
- Живые: тосты, success/aurora-flash, confetti, count-up чисел, hover-эффекты,
  ripple/press, shake при ошибке, копирование с micro-flash.
- Уважает `prefers-reduced-motion` (анимации отключаются).

## Назначение
Прототип — визуальный ориентир для реализации. Он соответствует
[дизайн-документации](../docs/design) (токены, компоненты, анимации, экраны) и
служит эталоном look &amp; feel для будущих сессий разработки.

> Это статичный прототип на HTML/CSS/JS: демонстрирует дизайн и поведение, но не
> подключён к бэкенду. Реальная логика — по [общей документации](../docs).
