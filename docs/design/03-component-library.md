# Design 03 — Библиотека компонентов

Каждый компонент описан по схеме: **назначение → анатомия → варианты → размеры →
состояния → поведение/анимация → a11y**. Состояния по умолчанию для всех
интерактивных: `default / hover / focus-visible / active(press) / disabled /
loading / error`.

---

## Кнопки (Button)

- **Назначение:** запуск действий.
- **Варианты:**
  - `primary` — заливка Aurora-градиентом, белый текст, glow-volt. Главное действие.
  - `secondary` — surface-2 фон, бордер, текст hi. Второстепенное.
  - `ghost` — прозрачная, текст, hover-подсветка. Третичное.
  - `outline` — бордер + прозрачный фон.
  - `danger` — заливка Coral, danger-glow. Опасные действия.
  - `success` — Mint. Подтверждения.
  - `icon` — квадратная, только иконка.
- **Размеры:** `sm` 32px / `md` 40px / `lg` 48px; padding по размеру; радиус md.
- **Состояния:**
  - hover: подъём -1px, усиление glow, sheen-блик.
  - active: scale 0.97, ripple от точки клика.
  - focus-visible: кольцо Volt + glow.
  - disabled: 45% opacity, без событий, cursor not-allowed.
  - loading: спиннер слева, текст приглушён, ширина фиксирована, клики блокированы.
- **С иконкой:** left/right icon, gap 8px; только-иконка = квадрат.
- **a11y:** `<button>`, `aria-busy` при loading, `aria-disabled`, минимум 40px тач-таргет.

## Поля ввода (Input / Textarea)

- **Анатомия:** label (плавающий), поле, вспомогательный текст, иконки (prefix/suffix),
  счётчик (для textarea).
- **Варианты:** text, email, password (toggle-глаз), number, search (иконка+очистка),
  textarea (auto-grow).
- **Состояния:** default, focus (бордер Volt + glow), filled, error (Coral бордер +
  сообщение + иконка), success (Mint), disabled, readonly.
- **Анимация:** плавающий label уезжает вверх при фокусе/заполнении (140ms).
- **a11y:** связка `label for`/`id`, `aria-invalid`, `aria-describedby` для ошибок.

## Селекты / выпадающие (Select, Combobox, Multi-select)

- **Select:** триггер-поле + меню, скользящая подсветка активного, scale-in меню.
- **Combobox:** ввод + фильтрация списка, подсветка совпадений.
- **Multi-select:** чипы выбранного внутри поля, удаление по клику.
- **Состояния:** open/closed, hover-опции, выбранная (галочка), disabled-опция.
- **a11y:** `role="listbox"`, навигация стрелками, Enter/Escape, `aria-expanded`.

## Чекбокс / Радио / Switch

- **Checkbox:** галочка «рисуется» (stroke-dash animation), Volt-заливка.
- **Radio:** точка scale-in.
- **Switch:** кружок толкается пружиной, трек меняет цвет (off серый → on Volt),
  micro-glow при включении.
- **a11y:** нативные инпуты/`role`, лейбл кликабелен, состояние по клавиатуре.

## Карточка товара (ProductCard)

- **Анатомия:** превью/иконка категории, бейдж наличия (В наличии N / Под заказ /
  Нет), заголовок, краткие атрибуты (гео, тип), цена (крупно), рейтинг, кнопка «Купить».
- **Состояния:** default, hover (подъём -4px, тень 3, градиентный бордер-glow, sheen),
  out-of-stock (приглушение + бейдж), loading (skeleton).
- **Анимация:** появление stagger fade-up; «Купить» → полёт в корзину.
- **a11y:** вся карточка — ссылка на товар; кнопка «Купить» — отдельный фокус.

## Бейджи и чипы (Badge / Chip / Tag)

- **Badge статуса:** paid=Mint, pending=Amber, cancelled=Coral, delivered=Beam;
  «в наличии» — с пульсирующей точкой.
- **Chip фильтра:** выбираемый, с крестиком удаления, активный — Volt-обводка.
- **Counter badge:** на иконке корзины/уведомлений, пружинит при изменении.

## Табы (Tabs)

- Горизонтальные, скользящий градиентный индикатор под активной вкладкой (layout
  animation), cross-fade контента. Вариант «pills» (заполненные).
- **a11y:** `role="tablist"/"tab"/"tabpanel"`, стрелки, `aria-selected`.

## Таблицы (DataTable, админка)

- **Анатомия:** заголовки (сортировка-стрелка), строки (hover-подсветка), выбор
  строк (чекбоксы), действия в конце строки, пагинация/бесконечная прокрутка снизу,
  тулбар (поиск, фильтры, bulk-действия) сверху.
- **Состояния:** loading (skeleton-строки), empty (иллюстрация + CTA), error (retry).
- **Отзывчивость:** горизонтальный скролл в контейнере, «липкий» первый столбец.
- **a11y:** семантические `<table>`, `scope`, сортировка объявляется `aria-sort`.

## Модалки и диалоги (Modal / Dialog / Confirm)

- **Типы:** confirm (да/нет), form-modal (форма), info, danger-confirm (доп.
  подтверждение для необратимого).
- **Поведение:** focus-trap, закрытие по Escape/бэкдропу (кроме критичных), возврат
  фокуса на триггер, блокировка скролла фона.
- **Анимация:** backdrop-fade + scale-in окна (spring), быстрый выход.

## Тосты / уведомления (Toast)

- **Типы:** success (Mint-glow), error (Coral), info (Beam), warning (Amber).
- **Поведение:** влет снизу-слева, стек до 3, авто-скрытие 4s с прогресс-полоской,
  пауза при hover, действие «Undo/Retry» опционально.
- **a11y:** `role="status"`/`aria-live="polite"` (ошибки — `assertive`).

## Дропдаун-меню (Menu) и тултипы (Tooltip)

- Menu: список действий, разделители, опасные пункты (Coral), иконки; scale-in из
  триггера. Tooltip: появление 300ms delay, fast-скрытие, стрелка к элементу.

## Навигация

- **Top App Bar:** логотип (лево), поиск (центр, раскрывается), баланс + профиль +
  язык + тема (право). Прозрачная поверх hero → уплотняется при скролле (blur+тень).
- **Sidebar (ЛК/админка):** пункты с иконками, активный — Volt-подсветка + индикатор,
  сворачиваемый (иконки-only), tooltip в свернутом виде.
- **Breadcrumbs:** путь навигации, разделители-шевроны.
- **Bottom Nav (mobile):** 4–5 иконок ключевых разделов.

## Прочие компоненты

- **Avatar:** инициалы/фото, статус-точка, размеры sm/md/lg.
- **Pagination:** страницы + prev/next, «…» для длинных.
- **Stepper:** шаги оформления/пополнения (active/done/upcoming), линия прогресса.
- **Progress / Meter:** линейный и круговой; count-up для чисел.
- **Empty State:** иллюстрация + заголовок + описание + CTA (пустая корзина/заказы/поиск).
- **Skeleton:** блоки-заглушки с shimmer под каждый тип контента.
- **Slider (range):** для фильтра цены, две ручки, подсветка активного диапазона.
- **Accordion / FAQ:** раскрытие высоты + поворот шеврона.
- **Rating:** звёзды (выбор при hover), полу-звёзды для среднего.
- **Code/Secret field:** моно-шрифт, маска «••••», кнопки показать/копировать/скачать,
  micro-flash при копировании.
- **QR block (пополнение):** QR + адрес + таймер + статус-индикатор (pulse).
- **File upload / Import (стоки):** drag-n-drop зона, прогресс, предпросмотр, ошибки строк.

## Правила консистентности

- Все интерактивные компоненты имеют полный набор состояний (см. верх).
- Фокус-видимость обязательна везде.
- Одинаковые радиусы/тени/тайминги из токенов — никакой самодеятельности.
- Каждый компонент имеет light- и dark-версию.
- Тач-таргеты ≥ 40×40px.
