# Design 02 — Система анимаций и flash-эффектов

Премиальная, «вызывающая», но управляемая моторика. Принцип: **эффектно, но
осмысленно** — каждая анимация подчёркивает действие, а не отвлекает. Всё уважает
`prefers-reduced-motion`.

## Принципы моторики

1. **Отзывчивость:** реакция на действие < 100ms (нажатие, hover).
2. **Направленность:** движение подсказывает, откуда/куда идёт элемент.
3. **Слоистость:** важные элементы двигаются чуть иначе (parallax/stagger).
4. **Свет как акцент:** glow/flash — награда за важные события (оплата, выдача).
5. **Сдержанность в массе:** flash-эффекты дозированы, чтобы не «мельтешило».

## Тайминги и кривые (из токенов)

| Токен | Значение | Когда |
|-------|----------|-------|
| instant 80ms | press feedback | нажатие кнопки, чекбокс |
| fast 140ms | hover, tooltip | наведение, мелкие изменения |
| base 220ms | большинство переходов | появление, смена состояний |
| slow 360ms | модалки, панели | входы крупных блоков |
| slower 600ms | hero, флеши | вступительные эффекты |

Кривые: `--ease-standard` (базовая), `--ease-emphasized` (акцент с овершутом),
`--ease-spring` (пружина для «живых» элементов), `--ease-out` (появления).

## Библиотека анимаций (каталог)

### 1. Появление элементов (enter)
- **fade-up:** opacity 0→1 + translateY 12px→0, 220ms ease-out.
- **scale-in:** opacity 0→1 + scale 0.96→1, 220ms emphasized.
- **stagger:** списки/сетки — задержка 40–60ms между элементами (каскад).

### 2. Hover / наведение
- Кнопки: подъём translateY -1px + усиление glow, 140ms.
- Карточки товара: translateY -4px + тень 2→3 + подсветка градиентного бордера.
- Ссылки/иконки: цвет + лёгкий beam-glow.
- **Sheen (блик):** диагональный градиент `--grad-sheen` проезжает по карточке/CTA
  за 600ms (эффект «стекла на солнце»).

### 3. Press / нажатие
- scale 0.97 + уменьшение тени, 80ms; отпускание — spring назад.
- **Ripple:** от точки клика расходится круг (Material-ripple) на filled-кнопках.

### 4. Focus (клавиатура)
- Кольцо фокуса: 2px Volt + внешний glow, анимированное появление 140ms.
- Никогда не убираем фокус ради «красоты» — a11y.

### 5. Flash / glow-события (сигнатура бренда)
- **Success flash:** при успешной оплате/выдаче — короткая вспышка Mint-glow по
  контуру карточки/экрана + «пульс» иконки-галочки (scale 1→1.15→1, 400ms spring).
- **Pulse-индикатор:** статус «в наличии»/«онлайн» — мягкая пульсация Beam.
- **Confetti/particles (дозированно):** при первой покупке/крупном событии —
  лёгкие частицы (можно отключить в настройках).
- **Aurora-shift:** фон hero/CTA — медленно смещающийся Aurora-градиент (8–12s loop).

### 6. Загрузка (loading)
- **Skeleton shimmer:** серые блоки с проезжающим бликом (1.2s loop).
- **Кнопка loading:** спиннер + блокировка + сохранение ширины (без «прыжков»).
- **Progress bar (top):** тонкая градиентная полоса вверху при навигации/запросах.
- **Inline-spinner:** для точечных подгрузок (24px).

### 7. Переходы между страницами (SPA)
- **Cross-fade + slide:** уходящая страница fade-out+slide 8px, входящая
  fade-in+slide, 300ms. Общий top progress bar.
- **Shared element (v2):** карточка товара «раскрывается» в страницу товара.

### 8. Микро-интеракции
- Переключатели (switch): «толкаемый» кружок со spring + смена цвета трека.
- Табы: подчёркивающий индикатор скользит между вкладками (layout animation).
- Аккордеоны: плавное раскрытие высоты + поворот шеврона.
- Копирование (данные аккаунта): иконка → галочка + toast + micro-flash.
- Добавление в корзину: товар «улетает» к иконке корзины, бейдж-счётчик пружинит.
- Числа/баланс: count-up анимация при изменении суммы.

### 9. Оверлеи
- **Модалка:** backdrop fade 220ms + окно scale-in 0.96→1 spring; выход быстрее (160ms).
- **Toast:** влет снизу-слева slide+fade, авто-скрытие 4s, прогресс-полоска.
- **Dropdown/Tooltip:** scale-in из точки триггера (origin transform), 140ms.
- **Drawer (моб. фильтры/меню):** slide с краю + backdrop.

## Матрица «действие → анимация» (примеры)

| Действие | Визуальная реакция |
|----------|--------------------|
| Наведение на CTA | подъём + glow + sheen-блик |
| Клик по CTA | ripple + press-scale + loading-состояние |
| Успешная оплата | success-flash по экрану + пульс галочки + toast + count-up баланса |
| Ошибка оплаты | shake поля/карточки (2 колебания) + danger-glow + toast |
| Добавить в корзину | «полёт» товара + пружина бейджа корзины |
| Открыть модалку | backdrop-fade + scale-in окна |
| Загрузка каталога | skeleton-shimmer карточек, затем stagger fade-up |
| Смена вкладки | скользящий индикатор + cross-fade контента |
| Копировать данные | иконка→галочка + micro-flash + toast «Copied» |
| Пополнение подтверждено | Aurora-flash + confetti (опц.) + обновление баланса count-up |

## Производительность и доступность

- Анимируем только `transform` и `opacity` (GPU), избегаем layout-триггеров.
- `will-change` — точечно и временно.
- Тяжёлые эффекты (particles, aurora) — ограничены по частоте и отключаемы.
- **`prefers-reduced-motion: reduce`** → отключаем движение/параллакс/particles,
  оставляем мгновенные fade без смещений; функциональность не страдает.
- Все интерактивные состояния имеют не-анимационный эквивалент (цвет/контур).

## Реализация (прод)

- CSS-переходы/`@keyframes` для базовой моторики (дёшево, надёжно).
- **Framer Motion** (React) для layout-анимаций, shared-element, stagger, springs.
- Particles/confetti — лёгкая canvas-библиотека или собственный минимальный эмиттер.
- Top progress bar — на события навигации/запросов (React Query + router).
