# Design 00 — Бренд-идентичность: AdVault

## Название

**AdVault** = **Ad** (advertising) + **Vault** (защищённое хранилище ценностей).

Считывается как «секьюрный премиум-сейф с рекламными аккаунтами». Ключевые
ассоциации: **надёжность, эксклюзивность, премиум, скорость доступа к активам**.

**Почему работает:**
- Короткое (2 слога), легко произносится на любом языке.
- «Vault» = доверие и безопасность — критично для площадки, где ходят деньги и аккаунты.
- Хорошо ложится в домен, логотип, иконку (сейф/щит/замок).
- Расширяемо: AdVault Pro, AdVault Agency, AdVault Wallet.

**Альтернативы (запасные, если основное занято):**
| Имя | Смысл |
|-----|-------|
| Admint | «минтинг»/выпуск свежих аккаунтов, свежесть |
| Adnova | «nova» — вспышка, премиум-энергия |
| Advolt | «volt» — заряд, скорость, flash-эстетика |
| Vaulta | производное от vault, мягче звучит |
| Adhaus | «дом» аккаунтов, солидность |

## Позиционирование и tone of voice

- **Позиционирование:** «Премиальный маркетплейс рекламных активов с мгновенной
  выдачей и гарантией».
- **Слоганы (EN, дефолт):**
  - Primary: *"Premium ad accounts. Unlocked instantly."*
  - Alt: *"Your vault of advertising power."*
- **Слоган (RU):** *«Премиальные рекламные аккаунты. Доступ за секунды.»*
- **Тон:** уверенный, лаконичный, технологичный, без «крика» и спама. Премиум =
  меньше слов, больше веса. Обращение на «вы» (RU), профессиональное «you» (EN).

## Логотип

**Концепция:** монограмма из буквы **«A»**, стилизованной под грань открытого
сейфа/щита, с диагональным «лучом-вспышкой» (flash), проходящим сквозь неё.

- **Логомарк:** ромбовидный щит с внутренней «A», угол среза даёт эффект скорости.
- **Логотип (wordmark):** `AdVault` — где `Ad` обычным весом, `Vault` — жирным,
  либо `V` подсвечена градиентом (signature gradient).
- **Иконка приложения / favicon:** щит-ромб с градиентной заливкой и глоу.
- **Клир-спейс:** отступ вокруг лого не меньше высоты буквы «A».
- **Минимальный размер:** логомарк — 24px, wordmark — 96px по ширине.

> Для прототипа логотип реализуется как inline-SVG/текст с градиентом и glow —
> внешние ассеты не требуются.

## Фирменные цвета (краткая палитра, полная — в 01-design-tokens)

Тёмная тема — основная («premium dark»), светлая — вторичная.

| Роль | HEX | Назначение |
|------|-----|-----------|
| Void (фон) | `#080A1A` | глубокий сине-чёрный фон |
| Surface | `#12152E` | карточки, панели |
| Volt (primary) | `#5B5BF6` | основной акцент, кнопки, ссылки |
| Pulse (accent) | `#E24BF0` | вспышки, выделения, hover-glow |
| Beam (accent) | `#22D3EE` | подсветка, индикаторы, неон |
| Mint (success) | `#2BD9A6` | оплачено/выдано |
| Amber (warning) | `#F5B740` | ожидание/под заказ |
| Coral (danger) | `#FF4D6D` | ошибки/отмена |
| Text hi | `#EEF1FF` | основной текст |
| Text lo | `#9BA1C9` | вторичный текст |

**Signature gradient («Aurora»):** `linear-gradient(135deg, #5B5BF6 0%, #9B4DF0 50%, #E24BF0 100%)`
— используется в лого, ключевых CTA, hero, активных индикаторах.

**Flash/glow-акценты:** свечение через многослойные `box-shadow`/`text-shadow`
на базе Volt/Pulse/Beam (см. 01 и 02).

## Типографика (бренд)

- **Display / заголовки:** геометрический гротеск — рекомендуется **Space Grotesk**
  или **Sora** (премиальный, техно-вайб). Жирные веса 600–800, плотный трекинг.
- **Body / интерфейс:** **Inter** — читаемость, полный набор языков (EN/RU).
- **Моно (данные аккаунтов, коды):** **JetBrains Mono** / **IBM Plex Mono**.
- В прототипе (self-contained, без внешних шрифтов) используется системный стек
  с плотным трекингом для display; в проде подключаются вебшрифты выше.

## Визуальный язык

- **Premium dark** с глубоким фоном и «стеклянными» (glassmorphism) поверхностями.
- **Aurora-градиенты** и **неоновые glow** на ключевых акцентах.
- **Flash-эффекты:** вспышки/шиммер при появлении, свечение на hover, «пульс» на
  важных статусах (успех оплаты, выдача).
- **Сетка и воздух:** несмотря на «вызывающий» вид — строгая сетка, чистая
  иерархия, ничего лишнего. Эффектность ≠ хаос.
- **Микро-детали:** тонкие градиентные бордеры, внутренние тени, зерно/noise-текстура
  фона (очень слабое), анимированные градиентные подложки в hero.

## Айдентика в цифрах (быстрый reference)

- Радиусы: 8 / 12 / 16 / 24 px (карты — 16, кнопки — 12, чипы — 999).
- Базовый ритм отступов: 4px-сетка (4/8/12/16/24/32/48/64).
- Тени: мягкие «ambient» + цветные «glow» на акцентах.
- Иконки: Lucide / Material Symbols, тонкая линия 1.5–2px.

Полные значения — в [01-design-tokens](./01-design-tokens.md).
