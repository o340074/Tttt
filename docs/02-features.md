# 02 — Функциональность

Приоритеты: **P0** — обязательно для MVP, **P1** — вторая волна (v2), **P2** — позже (v3).

## Витрина / каталог

| Фича | Приоритет | Описание |
|------|-----------|----------|
| Каталог с категориями | P0 | Дерево категорий, товары внутри. |
| Карточка товара | P0 | Название, описание, характеристики, цена, наличие. |
| Варианты товара (SKU) | P0 | Разные конфигурации: тип аккаунта, гео, лимит, тип выдачи. |
| Фильтры и сортировка | P0 | По категории, цене, гео, наличию. |
| Поиск | P0 | Полнотекстовый поиск по каталогу. |
| Индикатор наличия | P0 | «В наличии: N шт» / «Под заказ» / «Нет в наличии». |
| Отзывы и рейтинг | P1 | Отзывы только от купивших. |
| Рекомендации / похожие | P2 | Блок «с этим покупают». |

## Пользователь и аккаунт

| Фича | Приоритет | Описание |
|------|-----------|----------|
| Регистрация / вход | P0 | Email + пароль, JWT (access + refresh). |
| Подтверждение email | P0 | Ссылка активации. |
| Восстановление пароля | P0 | Через email-токен. |
| Личный кабинет | P0 | Баланс, заказы, купленные товары, профиль. |
| 2FA (TOTP) | P1 | Двухфакторная защита аккаунта. |
| OAuth-вход | P2 | Google / Telegram при желании. |

## Корзина, заказ, оплата

| Фича | Приоритет | Описание |
|------|-----------|----------|
| Корзина | P0 | Добавление нескольких позиций, изменение количества. |
| Оформление заказа | P0 | Проверка наличия, расчёт итога, создание заказа. |
| Оплата с баланса | P0 | Списание внутренних средств. |
| Пополнение баланса криптой | P0 | Крипто-эквайринг, вебхук подтверждения. |
| Прямая оплата заказа криптой | P1 | Без пополнения баланса, сразу за заказ. |
| Промокоды / скидки | P1 | Процент или фикс, лимиты, срок. |
| Оплата картой | P2 | Отдельная оценка рисков и провайдера. |

## Выдача товара

| Фича | Приоритет | Описание |
|------|-----------|----------|
| Авто-выдача из стока | P0 | Резерв + выдача единицы стока после оплаты. |
| Ручная выдача под заказ | P0 | Заявка → админ/поддержка выдаёт → покупатель получает. |
| Хранилище выданного | P0 | Купленные данные доступны в ЛК, копирование, скачивание. |
| Автозамена брака | P1 | Запрос замены в течение гарантийного окна. |
| Массовая загрузка стока | P0 | Импорт стоков из файла (CSV/TXT) в админке. |

## Поддержка

| Фича | Приоритет | Описание |

## Администрирование

| Фича | Приоритет | Описание |
|------|-----------|----------|
| CRUD товаров и категорий | P0 | Создание/редактирование, публикация. |
| Управление стоком | P0 | Загрузка, просмотр, статусы единиц. |
| Управление заказами | P0 | Просмотр, ручная выдача, отмена, возврат. |
| Управление пользователями | P0 | Поиск, блокировка, корректировка баланса. |

| Дашборд статистики | P1 | Выручка, заказы, топ-товары. |

| Аудит-лог | P1 | Кто что менял в админке. |
