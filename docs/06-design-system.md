# 06 — Дизайн-система (стиль Google Ads)

Цель — чтобы платформа с первого экрана читалась как **профессиональный продукт
Google**: чисто, светло, воздушно, с фирменными синими акцентами и Material-паттернами.

> ⚠️ Юридическая оговорка: вдохновляемся визуальным **языком** (Material Design,
> чистота, типографика), но **не копируем** логотипы, точные бренд-ассеты и не
> выдаём себя за Google. См. [09](./09-security-compliance.md).

## Принципы

1. **Чистота и воздух** — много белого пространства, крупные отступы.
2. **Иерархия** — заголовки, карточки, явные акценты на действиях.
3. **Доверие** — аккуратные тени, ровная сетка, отсутствие «кричащего» дизайна.
4. **Функциональность** — интерфейс как рабочий инструмент (dashboard-эстетика).
5. **Доступность** — контраст WCAG AA, фокус-состояния, клавиатурная навигация.

## Цветовая палитра

| Роль | Значение | Применение |
|------|----------|-----------|
| Primary (синий) | `#1A73E8` | кнопки, ссылки, акценты |
| Primary hover | `#1765CC` | ховер основных действий |
| Success (зелёный) | `#188038` | статусы «оплачено», «выдано» |
| Warning (жёлтый) | `#F9AB00` | «ожидание», «под заказ» |
| Danger (красный) | `#D93025` | ошибки, отмены |
| Text primary | `#202124` | основной текст |
| Text secondary | `#5F6368` | вторичный текст, подписи |
| Border / divider | `#DADCE0` | границы, разделители |

**Тёмная тема** (v2): инвертированные поверхности (`#202124` фон, `#8AB4F8` акцент).

## Типографика

- Шрифт интерфейса: **Roboto** (или системный `Inter` как fallback) — близко к
  Google Sans, свободно доступен.
- Заголовки: 28 / 22 / 18 px, вес 500–700.
- Основной текст: 14–16 px, вес 400.
- Подписи/лейблы: 12–13 px, вес 500, цвет secondary.

## Компоненты (базовый набор)

- **Кнопки:** filled (основное действие), outlined (второстепенное), text (третичное).
  Радиус 8px, высота 40px, чёткие фокус-состояния.
- **Карточки товаров:** белая поверхность, тень уровня 1, hover — тень уровня 2,
  бейдж наличия в углу, цена крупно.
- **Поля ввода:** outlined-стиль Material, плавающие лейблы, валидация под полем.
- **Навигация:** верхний app bar с логотипом, поиском по центру, балансом и
  профилем справа. Слева — sidebar в личном кабинете и админке.
- **Таблицы (админка):** плотные строки, сортировка, пагинация, фильтры сверху.
- **Бейджи статусов:** цветовая кодировка (см. палитру): paid=green, pending=yellow,
  cancelled=red.
- **Тосты/снекбары:** нижний левый угол, авто-скрытие.
- **Модалки/диалоги:** подтверждение оплаты, выдача, опасные действия.

## Ключевые экраны

1. **Главная / витрина** — герой-баннер, категории плитками, популярные товары.
2. **Каталог** — сетка карточек + панель фильтров слева, сортировка сверху.
3. **Карточка товара** — галерея/иконка, описание, выбор варианта (SKU), наличие,
   кнопка «Купить», отзывы.
4. **Корзина / оформление** — список позиций, итог, выбор способа оплаты.
5. **Личный кабинет** — сайдбар: Баланс, Мои заказы, Купленные товары, Тикеты, Профиль.
6. **Купленные товары** — данные к копированию/скачиванию, статус, кнопка «Заменить».
7. **Пополнение баланса** — сумма, выбор актива/сети, QR/адрес, статус ожидания.
8. **Тикеты** — список + чат-переписка.
9. **Админка** — дашборд, товары, стоки, заказы, пользователи, финансы, настройки.

## Адаптивность

- Мобильный-first сетки, брейкпоинты 640 / 768 / 1024 / 1280.
- App bar сворачивается в бургер, фильтры — в выезжающую панель.
- Таблицы админки — горизонтальный скролл в контейнере.

## Ассеты и иконки

- Иконки: **Material Symbols** (свободные) или Lucide.
- Свой логотип и название (**AdsMarket** или иное) — НЕ бренд Google.
- Иллюстрации: нейтральные, в Material-стиле.
