# AdsMarket — платформа продажи цифровых товаров (Google Ads аккаунты)

> **Статус проекта:** 🟡 Планирование. Код ещё не пишется — эта фаза посвящена
> проектированию продукта, архитектуры и подготовке подробной документации для
> следующих сессий разработки.

**AdsMarket** — профессиональная торговая площадка цифровых товаров, выполненная
в визуальном стиле Google Ads. Основной ассортимент — рекламные аккаунты Google Ads
разных типов (стандартные, агентские, с историей открутки и т.д.), а также другие
цифровые товары и услуги.

## 🎯 Ключевые решения (зафиксированы на этапе планирования)

- **Доставка товаров:** гибридная — авто-выдача из стока + ручная выдача под заказ.
- **Оплата:** криптовалюта (USDT/BTC/ETH через крипто-эквайринг) + внутренний баланс/кошелёк.
- **Стек:** React (фронтенд, отдельно) + Node.js/NestJS (бэкенд, отдельно) + PostgreSQL.
- **Языки:** мультиязычный интерфейс (i18n), язык по умолчанию — английский.
- **Стиль:** Google Ads / Material Design — чистый, светлый, синие акценты.
